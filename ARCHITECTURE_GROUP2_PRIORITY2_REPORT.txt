Изменения архитектуры:

1. price_list.xlsx
- Добавлены столбцы group2 и priority2.
- Добавлены служебные столбцы auto_select_rule и components, если их не было.
- Для общей группы ОАК используется group=cbc / priority.
- Для капиллярной/детской группы ОАК используется group2=cbcf / priority2.
- Приоритет 100 остается ручным несдвигаемым дефолтом.
- Автообучение работает только в диапазоне 10–90.

2. alias_groups.xlsx
- Добавлены alias для cbc: ОАК, общий анализ крови, клинический анализ крови, анализ крови.
- Добавлены alias для cbcf: ОАК из пальца, ОАК ребенку, ОАК детям, ОАК малышу, ОАК от 0 лет, ОАК капилляр, ОАК микровета и т.п.

3. pricing.py
- LabTest, CandidateItem и MatchedItem поддерживают group2/priority2.
- Поиск по alias_groups выбирает активную группу.
- Анализ может быть найден по group или group2.
- При сортировке вариантов используется активный приоритет конкретной группы.
- Добавлена функция learn_group_priority(active_group, selected_code).
- Самообучение:
  * не меняет 100;
  * выбранный анализ +1, но не выше 90;
  * остальные анализы группы -1, но не ниже 10;
  * меняет только priority или priority2 в зависимости от активной группы.

4. bot.py
- При выборе варианта через кнопку Изменить вызывается learn_group_priority().

5. order_service.py
- Улучшено отображение строк со статусом included.
